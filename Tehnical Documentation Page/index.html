Python

This section is the "meat" of the webpage; it will display an understanding of the code element in HTML.

In Python, like many other languages the first thing you may learn is how to print "Hello World" into the output. In Python it is very simple: 

                    
                        print("Hello World!")

Likewise, creating and calling a function in Python is also very simple: 

                    
                        def hello():
                            print("Hello World!")

                        hello()
Variables

In Python you dont need to declare the type of variable, it is inferred from the information it contains.

For Example, to create a string variable: 

                    
                        name = "Brandon"

An integer variable: 

                    
                        age = 22

As you can see it is very simple to create variables in Python.

Data Types

In programming there are seven different data types:

  • String - "Hello World"
  • Number - 22.5
  • Boolean - True or false
  • null - a value of nothing
  • undefined - an undefined value
  • Symbol - a symbol
  • Object - an object
